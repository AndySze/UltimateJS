BASIC SAMPLE DEMO
-------------------------------------------------------

Demonstrate basic features of the engine like:
- automatic cross-device resizing and scaling
- different application states
- gui
- pre-loading
- touch and mouse controls 
- sprites and animations


Working version link (run on any HTML5 compatible browser) 
http://www.logicking.com/UltimateJS/demos/BasicSample/
Compiled download version:
http://www.logicking.com/UltimateJS/downloads/BasicSampleDemo.zip


BasicSample demo should be complied via Ant Build and tested from the browaer
on your local HTTP server (for example EasyPHP).
